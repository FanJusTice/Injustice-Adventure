# 不义冒险（游戏项目）

## 🎮 项目介绍
《不义冒险》是一款横版 2D RPG 闯关类游戏。玩家将在充满挑战的世界中，扮演一位持剑冒险者，探索各类地图，击败敌人，揭示背后的真相。

## 🌟 游戏特色
- 🌍 多样的关卡与场景：从诡秘森林到地牢，层层关卡充满挑战。
- ⚔️ 实时战斗系统：轻击、多重攻击、滑铲与蹬墙跳组合形成爽快战斗体验。

## 🛠️ 使用技术
- 引擎：Unity（C#）
- 动画和光效：Unity Animator + DoTween
- UI系统：UGUI + Canvas
- 资产数据持久化：PlayerPrefs + Scriptable Object
- 资源管理系统：Addressables
- 版本管理：Git
